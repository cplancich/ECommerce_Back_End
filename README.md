# E-Commerce BackEnd

## Description
    
A Node / Express based back-end application used to read, create, update, and delete product data for an E-Commerce environment.
    
## Table of Contents
    
* [Installation](#installation)
* [Usage](#usage)
* [Questions](#questions)

## Installation
    
This application requires the use of Node.js, MYSQL shell, sequelize, and express.js to run correctly. Be sure to run an npm install command when cloning the repository; A full video walkthrough of installing and running the application can be found [here](https://drive.google.com/file/d/17X9VivXa-f8mW-EVge_An7vX9139G5BT/view?usp=share_link).
    
## Usage
    
Server configuration which can be used to create an inventory database for an E-Commerce web application.
